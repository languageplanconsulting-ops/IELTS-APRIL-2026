QUESTION 1 · TRUE / FALSE / NOT GIVEN
Many Madagascan forests are being destroyed by attacks from insects.
✗ ผิด · ไม่ได้ตอบ
FALSE
บทความให้ “สาเหตุที่ต่างออกไป” ซึ่งขัดกับโจทย์ → ตอบ FALSE
📌 ถ้าบทความ “ไม่พูดถึงสาเหตุเลย” ถึงจะเป็น NOT GIVEN — แต่นี่มันบอกสาเหตุอื่นที่ขัดกัน
Cagan · สอนทักษะที่เอาไปใช้ต่อได้ ไม่ใช่แค่เฉลยข้อนี้
โจทย์อ้างว่า
ป่าถูกทำลายโดย การโจมตีของแมลง (insects)
บทความบอกจริง ๆ · Para A
“Madagascar's forests are being converted to agricultural land for rice cultivation.”
ป่าถูกเปลี่ยนเป็นพื้นที่เกษตรเพื่อปลูกข้าว — โดยมนุษย์
บทความพูดถึงแมลง · Para B
Insects destroy the crops, forcing farmers to clear new forest.
แมลงทำลาย “พืชผล” (ไม่ใช่ป่า) → ชาวนาเลยถางป่าใหม่
ป่าถูกถางโดย มนุษย์เพื่อการเกษตร ไม่ใช่ถูกแมลงทำลายโดยตรง — คนละสาเหตุ = ขัดกับบทความ → FALSE
Tidwell · เทียบ claim ↔ หลักฐาน + Krug · ประโยคจริง ไม่ใช่คำคีย์เวิร์ด
⚠️
กับดักของข้อนี้: บทความ “มี” คำว่า insects จริง ทำให้หลายคนเผลอตอบ TRUE — แต่แมลงทำลาย พืชผล ไม่ใช่ ป่า. อย่าจับแค่คำตรงกัน ให้จับ ความหมาย
Brown · เข้าใจว่าทำไมนักเรียนถึงพลาด
คำศัพท์ที่ทำให้ข้อนี้ยาก
convert = เปลี่ยน (…ให้เป็น) cultivation = การเพาะปลูก clear (land) = ถางป่า

สิ่งที่เปลี่ยนจากของเดิม

  • คีย์เวิร์ดห้วน ๆ “forests converted agricultural land rice cultivation” → ประโยคจริงจากบทความ พร้อมไฮไลต์คำที่ขัดกัน
  • เพิ่ม กฎ FALSE vs NOT GIVEN ที่เอาไปใช้ข้ออื่นได้ (หัวใจของ T/F/NG)
  • ตัด note ที่ไม่เกี่ยว (“biodiversity loss… not all species are suffering”) ออก → ใส่ คำศัพท์ที่เกี่ยวจริง แทน
  • เพิ่ม “กับดัก” อธิบายว่าทำไมคนถึงตอบผิด
